# ShopSphere

Modern e-commerce frontend built with React, Redux Toolkit & SCSS

## Features
-Product listing with search filters
-Cart with quantity controls
-Redux Toolkit state management
-Fully responsive UI

## Tech Stack
- React(vite)
- Redux Toolkit
- React Router v6
- SCSS(sass)
- Fetch API
